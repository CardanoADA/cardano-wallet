Cardano.Wallet.BlockSyncer

Description

This module contains the ticking function that is responsible for invoking block acquisition functionality and executing it in periodic fashion.

Known limitations: - Blocks are produced by the network layer. They are not expected to produce any duplicates and to rollback.
