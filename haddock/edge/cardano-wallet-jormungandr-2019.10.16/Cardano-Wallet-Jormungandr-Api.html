Polymorphic wrapper type to put around primitive types and, 3rd party lib types to avoid defining orphan instances and/or, undesirable instances on primitive types. It helps to keep a nice separation of concerns between the API layer and other modules.

type GetBlockDescendantIds = "api" :> ("v0" :> ("block" :> (Capture "blockId" BlockId :> ("next_id" :> (QueryParam "count" Word :> Get '[JormungandrBinary] [BlockId]))))) Source #

Retrieve n descendants of a given block, sorted from closest to farthest.

There might also exist fewer than n descendants.

For n=3 we might have:

[genesis] ... -- [b] -- [b+1] -- [b+2] -- [b+3] -- ... -- [tip]
                  \       \                  \
                 parent    +--- descendants ---+
