Description

This module contains static configuration parameters. Rather than providing and carrying around a configuration file through the application, we resolve configuration data at runtime using the available environment.

This gives us a flexible and portable approach to software configuration, and remove some pain from the development perspective. Prior to starting, the wallet is expected to have a few configuration parameter available. One may rely on a `.env` file to bundle configuration settings together for a given target environment.
