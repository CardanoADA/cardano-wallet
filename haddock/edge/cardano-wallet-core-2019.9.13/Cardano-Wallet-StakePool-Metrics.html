Cardano.Wallet.StakePool.Metrics

Description

This module can fold over a blockchain to collect metrics about Stake pools.

It interacts with: - Cardano.Wallet.Network which provides the chain - Cardano.Wallet.DB - which can persist the metrics - Cardano.Wallet.Api.Server - which presents the results in an endpoint
