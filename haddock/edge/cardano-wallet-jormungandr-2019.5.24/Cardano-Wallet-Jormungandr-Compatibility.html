Description

Contains various implementation decision that are specific to a particular network / protocol. This allows us to easily select a particular backend (Byron, Shelley-Rust, Shelley-Haskell) and isolate the bits that vary between those backends.

KnownNetwork n => EncodeAddress (Jormungandr n) Source #

Encode an Address to a human-readable format. This produces two kinds of encodings:

  • Base58 for legacy / Byron addresses
  • Bech32 for Shelley addresses

The right encoding is picked by looking at the raw Address representation in order to figure out to which class the address belongs.

Decode text string into an Address. Jörmungandr recognizes two kind of addresses:

  • Legacy / Byron addresses encoded as Base58
  • Shelley addresses, encoded as Bech32
